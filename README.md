# CiCi

CiCi is a platform 

## Building locally

```sh
# Install `Node.js` version v20.7.0
nvm use

# Install `Yarn` version 4.0.2
corepack enable
```

### Build desktop script

```sh
# Install dependencies
yarn

#create app
cd apps
yarn create vite android-screen --template react-ts

```
